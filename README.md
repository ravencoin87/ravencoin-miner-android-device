# ravencoin-miner-android-device

RavenCoin CryptoMining for Android Devices

## Installation

### Download the Latest APK Release

You can download the latest version of the Ravencoin Miner for Android from the official releases page:

[Download app-release.apk](https://github.com/Ravencoin87/ravencoin-miner-android/releases/latest/download/app-release.apk)

### Installation Steps

1. **Download the APK**: Click the link above to download the latest `app-release.apk` file to your Android device or computer.

2. **Enable Unknown Sources** (if installing from outside Google Play Store):
   - Go to **Settings** → **Security** (or **Privacy**)
   - Enable **Unknown Sources** or **Install Unknown Apps**
   - Note: This setting may vary depending on your Android version and device manufacturer

3. **Install the APK**:
   - Navigate to your **Downloads** folder
   - Tap on the `app-release.apk` file
   - Follow the on-screen prompts to install the application

4. **Launch the Application**:
   - Open your app drawer and find "Ravencoin Miner"
   - Tap to launch the application
   - Configure your mining settings and wallet address

## Requirements

- Android 5.0 or higher
- Sufficient device storage space
- Stable internet connection
